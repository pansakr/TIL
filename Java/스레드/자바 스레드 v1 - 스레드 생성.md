### 스레드

* 하나의 프로세스 내에서 진행되는 작업의 단위

* 스레드 생성 방법 2가지

```
// Thread 클래스 상속
public class Thread1 extends Thread{

    @Override
    public void run() {
        for(var i = 0; i <= 20; i++){

            try {
                Thread.sleep(500); // sleep() - 인자로 주어진 시간만큼 스레드를 멈추는 메서드
            } catch (InterruptedException e) {
                throw new RuntimeException(e);
            }

            System.out.print(1);
        }
    }
}
```
```
// Runnable 인터페이스 구현
public class MyRunnable implements Runnable{
    @Override
    public void run() {
        for(var i = 0; i <= 20; i++){

            try {
                Thread.sleep(500);
            } catch (InterruptedException e) {
                throw new RuntimeException(e);
            }

            System.out.print(2);
        }
    }
}
```

* 만든 스레드를 실행시키는 메서드는 .start() 이다

* Run() 메서드에는 스레드로 실행할 로직을 구현하면 된다

```
..main(){

    Thread thread1 = new Thread1(); // Thread 상속시
    Thread thread2 = new Thread(new MyRunnable()); // Runnable 구현시

    // Runnable 익명 클래스로 생성
        Thread thread3 = new Thread(() -> {
            for (var i = 0; i <= 20; i++) {

                try {
                    Thread.sleep(500);
                } catch (InterruptedException e) {
                    throw new RuntimeException(e);
                }

                System.out.print(3);
            }
        });
}

    // 모두 메인 쓰레드에서 실행
    // 각각의 쓰레드를 만들었지만 메인 쓰레드만 사용하므로 여러 쓰레드를 만든 의미가 없다
//   thread1.run();
//   thread2.run();
//   thread3.run();

    // 각각 새로운 쓰레드를 생성하여 동시해 진행
    // 실행시마다 결과 다를 수 있다
    thread1.start();
    thread2.start();
    thread3.start();
```
