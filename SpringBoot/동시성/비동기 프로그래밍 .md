### 비동기 프로그래밍 

* 작업의 완료를 기다리지 않고 다른 작업을 동시에 진행하는 것

    - 구현하는 방법으로 멀티스레드 기반 비동기, 단일스레드 기반 비동기가 있음

    - 멀티스레드 기반 비동기
    
        - 하나의 요청에 대해 주 작업은 메인 스레도, 일부 작업은 별도 스레드에 위임해 처리하는 방식
        
        - 예시 : @Async, CompletableFuture, ExecutorService 등

    - 이벤트 루프 기반 비동기
    
        - 하나의 스레드로 여러 요청을 논블로킹으로 처리하는 방식
        
        - Spring WebFlux, Node.js 등

* 서버가 여러 사용자의 요청을 동시에 처리할 때 사용하는 멀티스레드 구조와는 다름

    - 멀티스레드 서버 구조

        - 여러 요청을 동시에 처리하기 위한 서버의 기본 설계 방식

    - 멀티스레드 기반 비동기 프로그래밍

        - 한 요청 내부의 여러 작업을 동시에 처리하거나, 기다리지 않게 만드는 프로그래밍 기법
