### 프로젝트 작성 순서

* 주제 선택

* 기능 구성

    - 구매하기, 장바구니 담기, 알림설정하기 등

* 사용자 흐름 구성

    - 프로젝트가 사용자 시점에서 어떤 순서로 진행되는지 도식화 

* 데이터 모델링

    - 필요한 기능과 사용자 흐름 구성에 매치되도록 테이블/컬럼 생성(ERD)

    - 테이블 사이의 관계와 컬럼의 제약 조건 등도 설정(ERD)

    - 작성한 ERD 를 바탕으로 DB 에 실제로 생성

* 애플리케이션 제작

    - 디렉토리 구성 방법 선택

    - DB와 매핑할 엔티티 클래스 먼저 작성

    - 깃허브와 연결
